## Terceira-Lua

Jogo de estratégia por turnos em Python onde dois lados disputam o futuro da humanidade após a chegada de uma misteriosa Segunda Lua. Agora, uma Terceira Lua se aproxima, trazendo novos conflitos, magia e consequências imprevisíveis.
O jogador poderá escolher entre diferentes personagens, cada um com habilidades, passivas e mecânicas exclusivas, enfrentando adversários controlados por inteligência artificial em batalhas estratégicas 1x1.

# Estrutura da Partida

* Combate 1x1.
* Cada jogador realiza apenas 1 ação por turno.
* Partidas rápidas, com duração aproximada de 3 a 5 minutos.
* Objetivo: reduzir a vida do adversário a 0.
